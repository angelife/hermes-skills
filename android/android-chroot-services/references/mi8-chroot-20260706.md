# Mi8 chroot 2026-07-06 现场记录

## Rootfs 来源
- 镜像站：`https://images.linuxcontainers.org/images/debian/bookworm/arm64/default/<date>/rootfs.tar.xz`
- 示例：`20260706_05%3A24/rootfs.tar.xz`
- 大小约 90MB（xz），解压后约 300MB tar
- Docker Hub 国内不可用时，这是最可靠的替代源

## 解压策略
- Mac 上：xz -dc rootfs.tar.xz > rootfs.tar
- 推手机后解压到独立临时目录
- tar 报 can't link 非阻塞，只要顶层目录齐全即可
- 最终目标：/data/local/tmp/debian

## resolv.conf symlink 坑
- 症状：写入报 No such file or directory
- 原因：默认 symlink -> /run/systemd/resolve/stub-resolv.conf
- 修复：rm 掉 symlink，写真实 resolv.conf
- 写文件推荐在 chroot 内部或 shell 用户直接操作

## Magisk/su 可写性不一致
- su -c cat > 失败；adb shell (shell 用户) 成功
- 可靠做法：脚本 push 后在 chroot 内部执行

## 路径约定
- chroot 根：/data/local/tmp/debian
- 数据：/data/alist, /data/alist-logs
- 自启：/data/local/tmp/chroot-start-services.sh -> /data/adb/service.d/
