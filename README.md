# Hermes Skills

A curated collection of **170+ reusable skills** for [Hermes Agent](https://hermes-agent.nousresearch.com) — an intelligent AI agent framework.

## What are skills?

Skills are procedural memory for AI agents — reusable workflows, commands, and best practices for recurring task types. Each skill is a `SKILL.md` file with YAML frontmatter and markdown body.
