## 常见问题与对策

| 问题 | 可能原因 | 对策 |
|------|----------|------|
| **搜索框索引变化** | 页面结构更新或捕获窗口切换导致索引不同 | 在每次捕获后重新确认索引；若不可靠，可使用 `web_search` 后手动打开浏览器标签页 |
| **权限弹窗** | macOS 系统弹出权限对话框（如可访问摄像头） | `computer_use` 操作前检查是否有弹窗；如出现未预期的弹窗，暂停并询问用户 |
| **搜索结果为广告或推荐** | YouTubesearch结果可能混入推荐内容 | 通过标题匹配过滤；如匹配度低，可在搜索框中加入更具体关键词（如 “OpenWrt”） |
| **标签为空导致定位困难** | AXButton/AXTextField 的 `label` 字段常为空 | 依赖 `AXStaticText` 或坐标定位；若不可靠，可使用 `web_search` 直接打开搜索页面 |
