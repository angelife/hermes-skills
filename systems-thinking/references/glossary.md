# 系统思考术语表

## A-C

- **适应力（Resilience）**：系统在遭受冲击或干扰后恢复和保持其基本功能的能力。通过多个调节回路和冗余路径实现。
- **存量（Stock）**：在任何时刻都能观察、感知、计数、测量的系统要素。是系统历史变化的记录和缓冲器。
- **层次性（Hierarchy）**：系统包含子系统、子系统包含更小子系统的结构。所有适应性良好的复杂系统都具有层次性。
- **次优化（Sub-optimization）**：子系统的目标凌驾于整体目标之上，导致整体绩效下降的现象。
- **参数（Parameter）**：系统中可调的具体数值（税率、补贴率、标准值等）。是最弱效力的杠杆点。
- **超越范式（Transcending Paradigms）**：最高杠杆点——意识到任何范式都只是观察世界的透镜之一，不执着于单一范式。

## F-J

- **反馈回路（Feedback Loop）**：当存量改变时，调节流入/流出量的信息回路。信息是反馈回路的核心。
- **范式（Paradigm）**：决定系统之所以为系统的心智模式——社会深层的信念、假设和世界观。
- **非线性（Nonlinearity）**：系统中投入与产出之间的非等比例关系。非线性导致"主导地位转换"。
- **杠杆点（Leverage Point）**：系统中施加一个小变化就能导致系统行为发生显著转变的位置。
- **公地悲剧（Tragedy of the Commons）**：共享的有限资源被不受限制的个体使用者在各自利益驱动下耗尽。
- **规则（Rules）**：定义系统范围、边界和自由度的激励、惩罚和限制条件。

## K-O

- **流量（Flow）**：改变存量的速率，分为流入量和流出量。
- **目标侵蚀（Drift to Low Performance）**：系统的实际表现逐渐变差，但人们的期望也随之降低，导致问题被忽视。
- **目标错位（Seeking the Wrong Goal）**：系统忠实地产出了错误的目标所对应的结果。
- **评估陷阱（Policy Resistance）**：系统中多个参与者有不同目标，各自把系统往不同方向拉，导致系统困在各方都不满意的状态。

## P-T

- **时间延迟（Time Delay）**：从决策到行动、或从行动到结果之间的延迟。每个存量都是一个延迟的来源。
- **上瘾结构（Addiction / Shifting the Burden）**：外部干预措施削弱了系统自身维持状态的能力，导致对干预的依赖越来越深。
- **事件（Event）**：系统表面可见的引述/发生，是最容易观察到但信息量最少的部分。
- **自组织（Self-Organization）**：系统使其自身结构更加复杂化的能力——学习、多样化、进化。
- **增强回路（Reinforcing Loop）**：强化系统原有变化态势的反馈回路，产生指数增长或指数崩溃。

## W-Z

- **系统（System）**：一组相互连接的事物，在一定时间内以特定行为模式相互影响，能够实现某个目标的整体。
- **系统基模（System Archetype）**：产生常见问题行为模式的典型系统结构模式。
- **有限理性（Bounded Rationality）**：系统中每个角色基于局部信息和有限能力做决策，个体理性加总可能违背系统整体利益。
- **政策阻力（Policy Resistance）**：系统中各个参与方因目标不同而产生的相互拉力，使系统难以向任一方向改变。
- **调节回路（Balancing Loop）**：使存量维持在目标值或可接受范围内的反馈回路，产生寻的或动态平衡行为。
- **竞争升级（Escalation）**：两个或多个参与者以对方状态为参照目标的增强回路，导致指数级升级（如军备竞赛）。

---

## 常用符号

| 符号 | 含义 |
|------|------|
| R | 增强回路（Reinforcing loop） |
| B | 调节回路（Balancing loop） |
| → | 因果关系/连接 |
| ☁ | 系统边界之外的开放存量（"云"） |
| □ | 存量 |
| →（带阀门）| 流量 |
