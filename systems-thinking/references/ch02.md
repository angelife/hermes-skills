# 第2章 系统大观园

> 具有相似反馈结构的系统会产生相似的动态行为，即使外部表现完全不同。

## 系统1.1：温度调节器（一个存量 + 两个相互制衡的调节回路）

**结构：**
- 存量：室温
- 回路 B1：火炉加热 → 升温至设定值
- 回路 B2：热量散失 → 向室外温度靠拢
- 两者对抗，室温稳定在设定值附近

**关键教训：**
- 需同时考虑所有重要流量，否则系统行为会出乎意料
- 信息反馈只能影响未来行为
- 时间延迟的存在（信息传递需要时间）

## 系统1.2：人口/经济系统（一个存量 + 一个增强回路 + 一个调节回路）

**结构（人口系统）：**
- 存量：人口
- 回路 R：出生率 → 人口增长 → 更多出生 → 指数增长
- 回路 B：死亡率 → 人口减少 → 调节平衡

**结构（经济系统——完全相同）：**
- 存量：实体资本
- 回路 R：投资 → 资本增长 → 更多产出 → 更多投资
- 回路 B：折旧 → 资本减少

> **核心洞见：** 人口和工业经济表面差异很大，但系统结构完全相同。具有相似反馈结构的系统产生相似的动态行为。

**可能的行为模式：**
1. 增长（R 主导）
2. 衰退（B 主导）
3. 稳定（两者平衡）
4. 波动（主导地位交替转换）

**系统分析的目的：** 不是预测会发生什么，而是探究在各种驱动因素变化时系统可能如何应对。

## 系统1.3：库存系统（含时间延迟的调节回路）

**结构：**
- 存量：库存量
- 目标：维持10天销售额的库存
- 关键：订货与到货之间存在**时间延迟**

**行为特征：** 当存在时间延迟时，库存量会随销售额变化而上下**振荡**。

**教训：**
- 调节回路上的时间延迟很可能导致系统的振荡
- 问题不一定是因为反应太慢，有时是因为**反应太快**（反应过度）
- 改变延迟的长短可以彻底改变系统行为
- 延迟常常是敏感的政策杠杆

## 系统2.1：受不可再生资源约束的系统

**结构：** 资本增长（增强回路）+ 有限资源（调节回路约束）
- 资本增长消耗不可再生资源
- 资源存量不断减少，最终限制资本增长
- 行为：先增长，到达峰值后下降（过冲-崩溃模式）

**重要事实：** 没有任何真实的物理系统可以永远增长下去。当一个成长系统出现时，必然存在限制增长的调节回路。

## 系统2.2：两个可再生性存量的系统（渔业/森林系统）

**结构：**
- 存量1：可再生资源（鱼/树）
- 存量2：资本（渔船/伐木设备）
- 回路 R：资本增长 → 更多捕捞 → 更多利润 → 更多投资
- 回路 B：资源存量变动对自然再生率的影响

**三种可能的行为模式：**
1. 可持续平衡
2. 过度开发 → 振荡 → 修复
3. 过度开发 → 崩溃（不可逆）

**核心教训：** 技术改进可能提高生产率，但同时也可能让系统陷入更严重的不稳定。
