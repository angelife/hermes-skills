# 邮箱 vs 远程控制 · 决策卡

用户常见跳跃：「给木设邮箱 = 就能远程控它」。

| 能力 | 木现状 | 说明 |
|------|--------|------|
| 远程下指令 | ✅ `mu` | OpenBridge 写 QwenPaw 对话页 |
| QQ Agent Mail 独立身份 | ❌ | 账号约 2 邮箱已满；木无本机 CLI 宿主 |
| 云端浏览器自动化 | ✅ | 沙箱 Chromium，非本机 |
| 控局域网/本机 | ❌ | 沙箱隔离 |

**先问有没有用：**
1. 要指挥 → `mu`，不必邮箱
2. 要公网收件身份 → 新微信账号/提额 + 合适宿主，或云端 IMAP/SMTP（仍非本机控权）
3. 要本机/局域网 → 土 / 金 / 水，不要派木
