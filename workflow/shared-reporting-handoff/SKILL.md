# Shared: 会话闭环与交接报告

## 适用 bot
土、金、水、火

## 触发条件
- 会话将要关闭 / 进入新会话
- 用户要求"总结"/"报告"/"交接"
- 任务完成后的状态存档

---

## 闭环报告模板

每个 bot 在会话结束时输出标准格式：

```
## [元素] · [角色] — 会话摘要

### 本次完成
- [x] 任务 1
- [x] 任务 2
- [ ] 未完成项

### 关键决策
- 决策 1（理由）
- 决策 2（理由）

### 遗留问题
- 问题 1（阻塞原因）
- 问题 2（建议后续方案）

### 环境状态
- 配置：有/无修改（附 md5）
- 进程：gateway PID / 状态
- 网络：IP / 代理状态
```

## 跨 bot 交接

当任务需要转给其他 bot 时：

1. **土同学** 接收任务 -> 分析 -> 决定分配给哪个 bot
2. 分配时附上：任务目标 + 已有上下文 + 约束条件
3. 接收 bot 完成后返回标准报告给土同学
4. 土同学汇总给用户
